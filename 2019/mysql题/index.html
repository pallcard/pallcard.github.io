Written by 刘科 with ♥ on June 1, 2019 in Mysql

1.左连接、右连接、内连接

左连接

以左表为基础，根据on后面的条件进行连接，若右表中无对应数据，则右表数据为NULL。

右连接

以右表为基础，根据on后面的条件进行连接，若左表中无对应数据，则左表数据为NULL。

内连接

两个表做笛卡尔积后根据on后面的条件进行筛选。
