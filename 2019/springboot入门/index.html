SpringBoot入门

Written by 刘科 with ♥ on June 8, 2019 in SpringBoot

创建工程

使用IDEA，按照提示创建即可。

修改Banner

在resources目录下创建banner.txt文件，改文件用于指定banner，文字生成banner的网址：http://patorjk.com/software/taag。

${AnsiColor.BLUE}
.____     ____  __.
|    |   |    |/ _|
|    |   |      <  
|    |___|    |  \ 
|_______ \____|__ \
        \/       \/
-----版本号-----${spring-boot.version}

说明： ${AnsiColor.BLUE} 表示Banner文字的颜色 ${spring-boot.version} 当前使用的SpringBoot版本

创建Controller类

@Controller
public class HelloWorld {
    @RequestMapping(value = "/hello",method = RequestMethod.GET)
    @ResponseBody
    public String hello(){
        return "Hello Spring Boot";
    }
}

访问：http://localhost:8080/hello，即可以看到Hello Spring Boot

另外一种注解方式

@RestController
public class HelloWorld {
    @GetMapping("/hello")
    public String hello(){
        return "Hello Spring Boot";
    }
}

说明 * @RestController：处理http请求：等同于@Controller+@ResponseBody * @GetMapping：@RequestMapping以get请求的简写

url的其他形式

@RestController
@RequestMapping("/test")
public class HelloWorld {
    @GetMapping(value={"/hello","/hi"})
    public String hello(){
        return "Hello Spring Boot";
    }
}
